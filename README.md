# WIRELESS SECURITY DETECTOR 
 Project Features
- 230VAC Supply voltage, optional 12VDC and 5VDC
- Wi-Fi and Bluetooth connection with ESP32 module
- Programmable interface
- Buzzer alarm
- RTC
- USB interface
- Two programmable relay outputs
- Motion sensor (PIR)
